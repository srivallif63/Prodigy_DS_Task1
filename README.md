# 📊 Prodigy InfoTech Data Science Internship - Task 01

## 🔍 Objective

The goal of this task is to visualize the distribution of a continuous or categorical variable using appropriate graphical representations such as **histograms** and **bar charts**.

---

## 📁 Dataset

The dataset contains population data of different countries across multiple years.
For this task, population data for the year **2020** is used.

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Visualizations

### 1️⃣ Histogram

* Displays the distribution of population values for the year 2020
* Helps in understanding how population is spread across countries

### 2️⃣ Bar Chart

* Shows the **Top 10 most populated countries (2020)**
* Useful for comparing population between countries

---

## 🚀 Steps Performed

1. Imported required libraries
2. Loaded the dataset using Pandas
3. Cleaned the data (handled missing values)
4. Filtered data for the year 2020
5. Created:

   * Histogram for population distribution
   * Bar chart for top 10 countries
6. Visualized results using Matplotlib & Seaborn

---

## 📌 Key Insights

* Population distribution is **highly skewed**, with few countries having very large populations
* Countries like **India and China** dominate the population count
* Most countries fall under lower population ranges

---

## 📷 Output

* Histogram of population distribution
* Bar chart of top 10 populated countries
